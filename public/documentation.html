Add a new user localhost:8080/users POST none A JSON object holding data about the user to be added, structured like this: 

          {
            "name": "Mary",
            "favoriteMovies": ""
          }

A JSON object holding data about the user that was added, including an ID, uniquely created by uuid.

Example:

          {
            "name": "Mary",
            "favoriteMovies": "",
            "id": "e0f166ca-a2e0-486a-92da-a8e0480762bd"
          }
Update user info (username) localhost:8080/users/ PUT [ID], example: 1 A JSON object holding data about the updated name of the user. Example: 
          {
            "id": 1,
            "name": "Kim Updated Name",
            "favoriteMovies": []
          }
A JSON object holding the updated data of the user. Example: 
          {
            "id": 1,
            "name": "Kim Updated Name",
            "favoriteMovies": []
          }
Add a favorite movie to favoriteMovies of the user. localhost:8080/users/ POST [ID]/[movieTitle]. Example: 1/Joker none A JSON object holding the data of the user including id, name and favoriteMovies. Example: 
          {
            "id": 1,
            "name": "Kim Updated Name",
            "favoriteMovies": [
              "Joker"
            ]
          }
